*O arquivo metodoGradiente executa a regressão linear utilizando o método de descida do gradiente. Logo, para executar o algorítmo basta executar o arquivo metodoGradiente ou chamar o script no console do octave.

*O arquivo metodoExato executa a regressão linear utilizando o método exato matricial. Este foi utilizado a fim de comparar ambas as abordagens e refinar os parametros para o método gradiente.

*O método de descida do gradiente foi implementado utilizando orientação a objetos. Essa característica só está presente no octave versão 4 ou superior.
